# Here are your Instructions

This project is a two‑sided local e‑commerce application (customer and seller panels). It now includes:

- A **user profile page** where customers and sellers can view and update their name, phone, address, and password.
- **Product image upload** for sellers: images are uploaded to **Cloudinary** cloud storage and delivered via CDN.
- **Docker support** for easy containerization and deployment.
- **GitHub Actions CI/CD** pipeline for automated testing and building.

For Cloudinary setup instructions, see [CLOUDINARY_SETUP.md](CLOUDINARY_SETUP.md).  
For deployment to production, see [DEPLOYMENT.md](DEPLOYMENT.md).
